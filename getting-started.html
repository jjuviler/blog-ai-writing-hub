Getting Started

What's in This Hub

The Blog AI Writing Hub has three types of resources for HubSpot blog writers:

  • Tools — standalone apps that plug into your existing workflow, like Blogbot (Google Docs add-on) and Copy Page Body (Chrome extension)
  • Trainings — guides and videos covering AI prompting methods and how to use Claude effectively
  • Claude Skills — shared AI skills that live inside Claude and activate automatically when relevant. They give Claude specialized knowledge and capabilities for HubSpot blog work

Step 1: Activate Your Claude Skills

All Claude Skills listed in this hub are shared with HubSpot blog writers — you don't need to download anything. To activate them:

  1. Open Claude and click Customize in the sidebar
  2. Go to the Skills tab
  3. Under Shared Skills, find the skill you want to enable
  4. Select it, then toggle it on in the top right

Skills available to you: Brief QA, Product Research Assistant, HubSpot Products, HubSpot Branding, and AEO Draft Auditor. Each page has details on what the skill does and when to use it.

Step 2: Install Blogbot

Blogbot is a Google Docs add-on that handles QA and CMS uploads for every post you write. Install it from the Google Workspace Marketplace — the Blogbot page has full setup and usage instructions.

Step 3: Learn the WRITE Framework

The WRITE Framework is the prompting methodology used across HubSpot blog writing. It structures your Claude prompts around five components — Who, Resources, Instructions, Terms, and Expected Outcome — and produces significantly better output than unstructured prompts. Worth learning before you start using Claude for outlines or drafts.
