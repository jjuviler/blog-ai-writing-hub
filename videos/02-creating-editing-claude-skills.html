Creating and Editing Claude Skills

FAQs

How do I install a Claude Skill?

  1. Download the .zip file from the Google Drive folder (linked on each skill page)
  2. Go to claude.ai → Settings → Capabilities → Skills
  3. Drag and drop the .zip file into the upload area
  4. Enable the skill in your skills list

Can I modify the skills?

Yes! Ask Claude: "Can you update the [skill name] to make the output more concise?" Claude can edit skills in real-time. Watch the video above to see how.

What if a skill doesn't work?

First, make sure it's enabled in your Skills settings. If it's still not working, reach out to Tim Biondollo (tbiondollo@hubspot.com) for troubleshooting.

Can I create my own skills?

Yes! Watch the video above for a full walkthrough. You can also check out how the HubSpot Artifact Creator was built using Claude's Skill Creator tool.

Can I suggest improvements or new skills?

Absolutely! If you have ideas for new skills or improvements to existing ones, reach out to Tim. All the skills started as suggestions from writers on the team.
