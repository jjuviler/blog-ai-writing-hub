Best Practices

Workflow Integration

Recommended Workflow for Listicle Posts

  1. Receive brief — Run Brief QA skill immediately
  2. Fix flagged issues — Update broken URLs and verify stale content
  3. Research products — Use Product Research Assistant with verification
  4. Create outline — Use WRITE Framework for structured prompting
  5. Write draft — Continue using WRITE Framework for each section
  6. Generate alt text — Use Alt Text Generator in Blogbot
  7. Review everything — Never skip human verification

Quality Control

Always Verify

  • URLs: Click every link to confirm it works
  • Pricing: Check official product pages for current rates
  • Product names: Verify against official HubSpot products page
  • Statistics: Trace claims back to original sources
  • Tone: Read aloud to ensure it sounds like HubSpot

Common Mistakes to Avoid

❌ Don't

Copy-paste AI output without review

✅ Do

Treat AI output as a first draft that needs your expertise

❌ Don't

Use skills without understanding what they do

✅ Do

Watch the video walkthroughs before using new skills

❌ Don't

Use AI to replace your expertise

✅ Do

Use AI to handle repetitive tasks so you can focus on strategy and insight
