HubSpot Branding Artifact Creator

Download from Google Drive

Overview

The HubSpot Branding Artifact Creator applies HubSpot's official branding — color palettes, logos, and design guidelines — to any artifact you create in Claude. Built using Claude's built-in Skill Creator tool, it's loaded with HubSpot brand templates so every report, slide deck, and visual output looks polished and on-brand.

This skill was created using Claude's built-in Skill Creator tool, fed with HubSpot brand templates, slide deck guidelines, color palettes, and logo assets. You can use the same approach to build your own custom skills.

What It Does

  • Brand Enforcement: Applies HubSpot color palettes, typography, and spacing to any artifact
  • Logo Integration: Adds the HubSpot logo in the correct position and size
  • Slide Deck Creation: Generates presentation-ready slide decks with HubSpot styling
  • Report Formatting: Structures monthly/quarterly reports with branded layout
  • Artifact Updates: Re-brands existing artifacts you've already created

When to Use

  • End-of-month or quarterly data reports
  • Presentations and slide decks for internal or external audiences
  • Any artifact you want to look polished and on-brand
  • Re-branding existing documents that don't use HubSpot styling
  • Situations where you need visually clean output quickly

Installation

  1. Download the skill: Click the button above to access the skill file
  2. Go to Claude: Navigate to claude.ai → Search → Customize
  3. Open Skills tab: Click the Skills tab in the Customize panel
  4. Upload: Click the + button → "Upload Skills" → select the file

How to Use

  1. Open a new Claude chat
  2. Create your content — a report, data summary, presentation outline, etc.
  3. Add "with HubSpot branding" to your prompt — Claude will fire the skill automatically
  4. Review the branded output and iterate with Claude as needed
  5. Export to Google Slides or download as needed

Example Prompts

Update my teaser artifact with HubSpot branding.
Run my February community report with HubSpot branding.
Create a slide deck summary of this data with HubSpot branding.

Tips & Best Practices

  • Add "with HubSpot branding" to any artifact creation prompt — the skill fires automatically
  • Iterate with Claude — ask it to adjust colors, layout, or add/remove sections
  • Export to Google Slides to make further edits in a familiar tool
  • Don't assume the output is final — everything is editable
  • Make sure the skill is active (enabled in your Skills list) before prompting

Creating Your Own Skills

You can build custom skills using the same approach used to create this one:

  1. Go to claude.ai → Search → Customize → Skills tab
  2. Click "Skill Creators" to see examples
  3. Click "Try in Chat" or "Create with Claude"
  4. Claude will guide you through the process with questions
