AEO Draft Auditor

Download Skill

Overview

The AEO Draft Auditor scans blog drafts against HubSpot's official AEO source-of-truth documents, flagging terminology misalignment, incorrect product naming, and inaccurate claims before they make it to publish.

HubSpot's AEO launch has specific terminology rules, positioning requirements, and language that must be used consistently across all blog content. Manual cross-referencing is slow and error-prone — this skill automates that check.

Reference Documents

The skill audits against three official AEO source documents:

  1. Positioning & Messaging Guide SSOT
  2. AEO FAQ
  3. AEO Editorial & Terminology Guidelines

What It Audits

  • Terminology & Product Naming: Flags use of banned terms or incorrect product names
  • Product Accuracy: Verifies claims about AEO features against official documentation
  • Stats, Proof & Competitive Claims: Checks data points and competitive assertions
  • CRM & Echoes Language: Flags violations of CRM-specific language rules
  • Messaging & Positioning Alignment: Ensures the draft aligns with official AEO positioning

Severity Levels

🔴 Must Fix

  • Factual errors about AEO capabilities
  • Banned terminology or product names
  • Wrong pricing or feature claims
  • CRM language violations
  • Using "standalone" in external content
  • Positioning AEO as an SEO replacement

🟡 Recommended Fix

  • Suboptimal phrasing that could be stronger
  • Using "questions" instead of "prompts"
  • Vague "AI-powered" claims without specificity

When to Use

  • Before publishing any AEO-related blog content
  • After drafting — before sending to editor review
  • When editing freelancer-submitted AEO content
  • Any time you need to verify AEO terminology compliance

Installation

  1. Download the skill: Click the button above to access the skill package
  2. Open Claude Settings: Go to claude.ai → Settings → Capabilities → Skills
  3. Install the skill: Drag and drop the zip into the Skills upload area
  4. Activate the skill: Enable "AEO Draft Auditor" in your skills list

How to Use

  1. Paste your complete AEO blog draft into Claude
  2. Tell Claude: "Audit this draft using the AEO Draft Auditor skill"
  3. The skill auto-fetches the latest AEO reference documents
  4. Claude scans the draft and returns a structured audit report
  5. Address all 🔴 Must Fix issues before submitting for review

Example Audit Report Format

AEO DRAFT AUDIT REPORT
======================
🔴 MUST FIX (3 issues)
- Para 2: "AEO replaces SEO" — AEO should complement SEO, not replace it
- Para 5: Uses banned term "standalone AEO" — remove "standalone"
- Para 8: Pricing claim unverifiable — remove or verify against live page

🟡 RECOMMENDED FIX (2 issues)
- Para 1: "questions" should be "prompts" per AEO terminology guide
- Para 6: "AI-powered" is vague — specify what AI capability is being referenced
